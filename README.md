# 이 프로젝트는 서울사이버대학교 과제 제출용으로 만들었습니다.

## 제작에 사용한 SW :

가. 개발 SW : Intellij

나. 서버 : 무료 Oracle-cloud Linux Server

다. SQL / WAS SW : Mysql8 / tomcat

## 게시판 개요
가. 게시판 작성간 확장성을 고려(차후에 본인이 추가하여 사용)하여 만들었습니다.

나. 게시판은 두가지 종류로 제작했습니다.
  - 개인만 작성하고 수정할 수 있는 메모게시판
  - 모두가 사용할 수 있는 '카테고리'가 있는 게시판

다. 게시판 사용의 보안성을 위해서 '가입철차'를 만들어 회원제 홈페이지로 제작했습니다.

라. 게시판의 공통 기능 : 
  - 게시글 : 작성 / 수정 / 삭제
  - 파일 : 업로드 / 변경 / 삭제 (단, 테스트 프로젝트이므로 업로드가능 용량은 1GB로 제한함)
  - 댓글 : 추가 / 수정 / 삭제

마. 메모장 게시판의 특징 : 관리자라고 하더라도, 내가 쓴 글은 다른사람이 확인 할 수 없음.

바. 회원관리
  - 원할한 게시판 운영을 위해서 회원 가입 / 관리가 가능하고
  - 가입자별 개별 권한을 부여 할 수 있음
  - 게시판 관리자 권한 부여시 일반 게시판(메모장 제외)의 게시글 수정 / 삭제 가능함(일반 읽기/쓰기 권한자는 자신의 게시글(댓글포함)만 수정 / 삭제 가능함)
  - 개인의 개인정보나 비밀번호 변경을 위해 별도의 변경할 수 있는 항목을 제작함.

사. 게시판에 보이지는 않지만, DB와의 접속간 주소 / ID / 비밀번호는 암호화하여 접속
